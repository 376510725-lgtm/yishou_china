hde# 益寿巴渝后台管理系统

## 📋 业务与功能

### 1.1 核心目标

为「益寿巴渝」APP 平台提供完整的 B 端管理后台，覆盖用户、组织、内容、圈子、服务、活动、权限等全业务链路管理。支持平台方自运营，同时支持向合作企业开放受控权限，实现多租户管理能力。

### 1.2 功能清单

- **数据概览（Dashboard）**：平台核心 KPI 指标卡（含图标+趋势箭头）+ 近30日 ECharts 趋势折线图
- **用户管理**：用户列表（高级筛选 + 批量操作）、用户详情全页面（6 Tab：基本信息/作品管理/社交关系/积分记录/登录日志/封禁记录）、重置密码、封禁/解封操作
- **组织管理**：组织树层级、成员管理、对应 APP「我的组织」功能
- **内容管理**：图文/视频 Tab 切换列表、统计卡片（待审核/今日已审核/通过率/累计）、高级筛选（状态/作者/时间/关键词）、批量操作（批量通过/拒绝）、内容详情 Modal（含审核历史）、拒绝原因必填
- **圈子管理**：卡片式圈子列表（含成员/帖子/活动统计）、点击进入圈子详情页（含圈子活动列表 + 管理员管理 Tab）、禁用/启用
- **服务管理**：
  - **服务订单**（方案A：纯订单管理）
    - 顶部4列统计卡片：总订单数、待处理、进行中、已完成
    - 8类服务类型 Tab 切换（全部/助餐/助浴/助洁/助医/助急/助行/疗养/活动）
    - 筛选条件：订单状态（全部/待处理/进行中/已完成/已取消）、时间范围、关键词搜索
    - 订单列表：序号、订单编号、服务类型、服务商、用户、手机号、服务地址、预约时间、金额、状态、操作
    - 订单详情 Modal：订单信息 + 用户信息 + 服务商信息 + 状态操作（开始处理/确认完成/取消）
  - **服务商管理**（方案A：纯服务商管理）
    - 顶部3列统计卡片：服务商总数、正常运营、已停用
    - 筛选条件：服务商状态、服务类型、关键词搜索
    - 服务商列表：序号、服务商名称、服务类型、联系方式、服务区域、评分、订单数、入驻时间、状态、操作
    - 新增/编辑/查看 Modal：服务商名称、服务类型、联系方式、服务区域、入驻时间、评分、订单数、状态
- **活动管理**：平台级活动列表（含开始/结束时间、地点、报名情况、状态）、新增/编辑（含参与范围：全平台/按区域/按组织/按圈子）、草稿/发布/下架状态流转、活动详情 Modal、筛选条件（状态/参与范围多选/时间范围/关键词）
- **积分管理**：积分流水记录（获得/消耗/手动发放）+ 积分规则配置（启用/停用/编辑）、统计卡片（总记录/总获得/总消耗/活跃用户）、筛选条件（类型/用户/时间范围/关键词）
- **消息管理**：推送记录、新建推送（推送对象：全平台/按区域/按组织/按圈子）、筛选条件（推送对象多选/状态/时间范围/关键词）、状态流转（保存→待发送→已发送/已取消）、操作列（发送/取消）、创建人字段
- **举报管理**：举报列表分类处理（内容/用户/圈子 Tab）、统计卡片（总举报/待处理/已处理/处理率）、筛选条件（状态/时间范围/关键词）
- **权限管理**：角色权限树配置 + 企业账号管理、统计卡片（总账号/启用账号/角色数量/活跃率）、筛选条件（角色多选/状态/关键词）
- **登录页**：Logo + 账号密码/手机号密码 Tab，不支持注册

### 1.3 交互要点

- **用户管理**：
  - 列表视图：高级筛选栏（搜索/状态/性别/地区/注册时间范围）、批量操作（批量封禁/解封）、表格支持多选
  - 详情视图：全页面展示，返回按钮回到列表，6 个 Tab 切换
    - 基本信息：账号信息 + 个人资料 + 第三方授权（支持解绑）
    - 作品管理：统计摘要 + 筛选（类型/状态/关键词）+ 卡片式列表（含封面占位、互动数据、操作按钮）
    - 社交关系：左右两列展示关注/粉丝列表，支持跳转查看
    - 积分记录：当前余额大数字 + 流水表格 + 手动发放积分按钮
    - 登录日志：登录时间/设备/IP/地点表格
    - 封禁记录：Timeline 展示封禁历史，封禁中可解封
  - 操作：重置密码 Modal、封禁/解封 Popconfirm、作品详情 Modal、手动发放积分 Modal
- 登录页：Tab 切换「账号登录」/「手机号登录」；登录后进入主界面
- 主布局：左侧折叠侧边栏 + 顶部 Header（面包屑 + 用户头像 + 退出）+ 内容区
- 侧边栏激活项橙色高亮
- 详情用全页面（用户）或 Drawer（其他），新增/编辑用 Modal，删除/禁用用 Popconfirm 二次确认
- 组织管理：左树右表布局，点击组织节点显示成员列表
- **内容管理**：
  - 顶部统计卡片：待审核数量（橙色高亮）、今日已审核、今日通过率、累计审核总量
  - 高级筛选栏：审核状态（全部/待审核/已通过/已拒绝）、作者搜索、发布时间范围、关键词搜索、重置按钮
  - 表格展示：ID/类型图标/标题（可点击查看详情）/作者（可点击跳转用户详情）/发布时间/互动数据（点赞/评论/浏览）/状态/操作
  - 批量操作：表格支持多选（仅待审核可选）、底部浮动操作栏（批量通过/批量拒绝/取消选择）
  - 内容详情 Modal：标题+类型+状态标签、作者信息、内容区、互动数据统计、审核历史（已审核内容显示审核时间/审核员/拒绝原因）
  - 拒绝原因 Modal：常用原因快捷选择（内容违规/涉及敏感信息/低质量内容/虚假信息/其他）、拒绝原因输入框（必填，至少10字）
  - 操作流程：单条通过（Popconfirm 二次确认）、单条拒绝（打开拒绝原因 Modal）、批量通过（Popconfirm 确认）、批量拒绝（统一填写拒绝原因）
- **活动管理**：
  - 新增/编辑含「参与范围」字段（全平台/按区域/按组织/按圈子）、活动时间（开始/结束时间）、地点、最大报名人数、活动简介、宣传图片（可选，最多3张）
  - 参与范围选择：Radio.Group 单选，选择「按区域」显示 Cascader 区域选择器，选择「按组织」显示 Transfer 穿梭框多选组织，选择「按圈子」显示 Select 多选圈子
  - 高级筛选栏：状态筛选（全部/草稿/已发布/已下架）、参与范围筛选（多选：全平台/按区域/按组织/按圈子）、参与范围详情筛选（多选：具体区域/组织/圈子）、时间范围筛选、关键词搜索、重置按钮
  - 活动列表：ID/活动名称（可点击查看详情）/活动时间（开始~结束）/地点/参与范围/报名情况（进度条）/状态/操作
  - 活动详情页：全页面展示，返回按钮回到列表，2 个 Tab 切换
    - 活动信息：基本信息卡片（时间/地点/参与范围/创建人/创建时间/发布时间）+ 报名情况（进度条）+ 活动简介 + 宣传图片
    - 报名成员：成员列表表格（用户昵称/头像/手机号/所在地区/报名时间/状态/操作）
  - 状态流转：草稿→发布、已发布→下架、已下架→重新发布
  - 保存选项：保存为草稿、保存并发布
  - 操作按钮：草稿状态（编辑/发布/删除）、已发布状态（编辑/下架）、已下架状态（重新发布/删除）
- 圈子活动：表格顶部可按圈子下拉筛选；新建时先选圈子，再填写活动信息
- **圈子管理**：
  - 列表视图：筛选栏（分类/状态/排序/关键词）、4列卡片网格（缩小尺寸，含封面/统计数据）、批量操作（可选）
  - 详情视图：全页面展示，返回按钮回到列表，5 个 Tab 切换
    - 基本信息：圈子信息展示 + 管理员列表表格
    - 成员管理：统计摘要（总成员/本周新增/活跃/沉默）+ 筛选栏 + 成员表格（头像/昵称/角色/加入时间/发帖数/最后活跃/状态/操作）
    - 帖子管理：统计摘要（总帖子/今日新增/正常/已删除）+ 帖子表格（类型图标/标题/作者/发布时间/互动数据/状态/操作）
    - 活动管理：高级筛选栏（审核状态/活动状态/所属圈子/时间范围/关键词）+ 活动列表表格（已移除新建活动按钮）
    - 数据统计：趋势图表展示（待完善）
  - 操作：点击卡片进入详情、点击成员/作者跳转用户详情、设为/移除管理员、移出圈子、置顶/取消置顶帖子、删除帖子
- **圈子活动审核**：
  - 活动列表：ID/活动名称（可点击查看详情）/所属圈子（可点击跳转）/活动时间（开始~结束）/地点/报名情况（进度条）/审核状态/活动状态/操作
  - 高级筛选栏：审核状态（全部/待审核/已通过/已拒绝）、活动状态（全部/即将开始/进行中/已结束）、所属圈子下拉、时间范围、关键词搜索、重置按钮
  - 活动详情 Modal：标题+状态标签、基本信息（所属圈子/时间/地点/创建人）、报名情况（进度条）、活动简介、宣传图片、审核历史（Timeline）
  - 审核操作：待审核状态显示通过/拒绝按钮，已通过状态显示编辑/删除按钮，已拒绝状态显示删除按钮
  - 拒绝原因 Modal：常用原因快捷选择（活动信息不完整/时间冲突/内容不符合规范/涉及敏感内容/其他）+ 自定义输入（必填，至少10字）
  - 新建/编辑 Modal：活动名称/所属圈子/开始结束时间/地点/最大报名人数/活动简介/宣传图片（可选，最多3张）

- **服务管理**：
  - **服务订单**（独立页面）：统计卡片（总订单数/待处理/进行中/已完成）+ 服务类型Tab（全部/助餐/助浴/助洁/助医/助急/助行/疗养/活动）+ 筛选条件（订单状态/时间范围/关键词）+ 订单列表（订单编号/服务类型/服务商/用户/手机号/地址/预约时间/金额/状态）+ 订单详情Modal（订单信息+用户信息+服务商信息+状态操作）
  - **服务商管理**（独立页面）：统计卡片（服务商总数/正常运营/已停用）+ 筛选条件（服务商状态/服务类型/关键词）+ 服务商列表（序号/名称/服务类型/联系方式/服务区域/评分/订单数/入驻时间/状态）+ 新增/编辑/查看Modal
- **积分管理**：
  - 顶部 Tab 切换「积分记录」/「积分规则」
  - 统计卡片：总记录数、总获得积分、总消耗积分、活跃用户数
  - 积分记录筛选：类型筛选（全部/获得/消耗/手动发放）、用户搜索、时间范围筛选、关键词搜索、重置按钮
  - 记录页支持按用户/原因搜索
  - 支持手动发放积分（选用户+输入积分+填写原因）
- **消息管理**：
  - 新建推送对象支持：全平台/按区域/按组织/按圈子四种维度
  - 推送对象选择：Radio.Group 单选，选择「按区域」显示 Cascader 区域选择器，选择「按组织」显示 Transfer 穿梭框多选组织，选择「按圈子」显示 Select 多选圈子
  - 筛选条件：推送对象筛选（多选：全体用户/各区域/各组织/各圈子）、状态筛选（全部/待发送/已发送/已取消）、时间范围筛选、关键词搜索、重置按钮
  - 状态流转：保存→待发送→已发送/已取消
  - 表格字段：ID/推送标题/推送对象/创建人/创建时间/发送时间/状态/操作
  - 操作列：待发送状态显示「发送」和「取消」按钮，已发送/已取消状态无操作按钮
  - 新建推送：点击保存后状态默认为「待发送」，不直接发送
- **举报管理**：
  - 统计卡片：总举报数、待处理数、已处理数、处理率
  - Tab 切换：内容举报/用户举报/圈子举报
  - 筛选条件：状态筛选（全部/待处理/已处理）、时间范围筛选、关键词搜索（举报对象/举报人/原因）、重置按钮
  - 快速处理操作：查看详情、标记已处理
- **权限管理**：
  - 统计卡片：总账号数、启用账号数、角色数量、活跃率
  - Tab 切换：角色管理/账号管理
  - 账号管理筛选：角色筛选（多选）、状态筛选（全部/启用/禁用）、关键词搜索（账号/企业）、重置按钮
  - 角色 Tab：角色列表 + 编辑权限（权限树 Drawer）
  - 账号 Tab：企业账号列表 + 新增账号 Modal
- 退出登录：头像下拉 → 退出 → 返回登录页
