# 益寿服务端 APP - 设计文档

> **版本**: v1.0
> **日期**: 2026-06-04
> **说明**: 面向服务商和服务人员的双角色移动端APP原型，采用深空暗夜科技感设计风格

---

## 1. 设计决策

| 维度 | 方案 | 说明 |
|------|------|------|
| 展示方式 | 单框架 + 下拉切换 | 顶部下拉选择角色，下方显示手机框架 |
| 设计风格 | 深空暗夜科技感 | 深色背景 + 霓虹渐变 + 玻璃拟态 |
| 配色策略 | 角色区分配色 | 服务商蓝 / 服务人员紫 |
| 变体选择 | 标准深空 | 玻璃卡片 + 柔和渐变 + 细线条图标 |

## 4. 注册流程

### 服务商注册流程

```
注册页 → 入驻介绍页 → 填写信息 → 上传资质 → 申请成功
```

| 步骤 | 页面 | 说明 |
|------|------|------|
| 1 | 注册页 | 输入手机号获取验证码，设置密码 |
| 2 | 入驻介绍页 | 展示入驻优势和条件，点击"立即入驻" |
| 3 | 填写信息 | 服务商名称、联系人、电话、地址、服务类型 |
| 4 | 上传资质 | 营业执照、身份证等（拍照或相册） |
| 5 | 申请成功 | 提示"资料已提交，请等待审核" |

### 服务人员注册流程

| 步骤 | 页面 | 说明 |
|------|------|------|
| 1 | 注册页 | 输入手机号、邀请码、验证码、密码 |
| 2 | 注册成功 | 跳转到服务人员首页 |

### 人员管理布局

人员管理

### 任务列表布局

```
我的任务
├── 顶部统计（今日任务、进行中、已完成）
├── Tab（全部/待接单/进行中/已完成）
├── 任务卡片列表
│   ├── 服务类型标签
│   ├── 用户信息（头像+昵称）

│   ├── 预约时间
│   ├── 订单金额

```

### 任务状态机

```
待接单 → 已接单 → 服务中 → 已完成
   ↓         ↓
 已拒绝    已取消
```

## 8. 组件设计规范

### 按钮

| 类型 | 样式 |
|------|------|
| 主按钮 | 渐变背景 + 白色文字 + 圆角28px + 高度56px |
| 次按钮 | 透明背景 + 边框 + 文字色 + 圆角28px |
| 危险按钮 | 红色背景 + 白色文字 |

### 卡片

| 属性 | 值 |
|------|------|
| 背景 | `rgba(255,255,255,0.05)` |
| 边框 | `1px solid rgba(255,255,255,0.1)` |
| 圆角 | 16px |
| 内边距 | 16px |

### 输入框

| 属性 | 值 |
|------|------|
| 背景 | `rgba(255,255,255,0.08)` |
| 边框 | 无 |
| 圆角 | 8px |
| 高度 | 48px |
| 聚焦状态 | 边框变为角色主色 |

### 标签（Tag）

| 属性 | 值 |
|------|------|
| 背景 | 角色主色15%透明度 |
| 文字色 | 角色主色 |
| 圆角 | 4px |
| 内边距 | 2px 8px |
